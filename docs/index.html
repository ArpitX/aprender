Aprender

A responsive web template

Overview

Aprender is a front-end template that helps you build fast, modern, responsive websites. It is designed to be used for teaching a course or for Documentation purpose. It is a single page template i.e it doesn't need to loaded again when links for different pages are opened.

Installation

Steps

  • Get a copy of the code.
  • Install the dependencies if you don't already have them.
    npm install
  • Install gulp locally on project folder
    npm install --save-dev gulp
  • To run the server, type in terminal or command prompt:
    gulp serve
  • Modify the application to your liking.
  • Deploy your production code.
